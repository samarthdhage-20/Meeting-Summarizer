# 🎤 AI Meeting Summarizer

An AI-powered Meeting Summarizer built using **Node.js**, **Express.js**, **Groq Whisper**, and **Llama 3.3**.

The application converts meeting audio into an accurate transcript and automatically generates:

- Executive Summary
- Key Decisions
- Action Items
Video Link- https://drive.google.com/file/d/1931gWVD75wwT3M-WQ0wUtN_2OmLvF28I/view?usp=sharing
---

## 🚀 Features

- Upload meeting audio
- Automatic Speech Recognition (Groq Whisper)
- AI-generated Executive Summary
- Key Decision Extraction
- Action Item Generation
- Copy Summary
- Download Report
- Modern Responsive Dashboard

## ⚙ Installation

Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/Meeting-Summarizer.git
```

Install packages

```bash
npm install
```

Create `.env`

```env
PORT=3000

GROQ_API_KEY=YOUR_API_KEY
```

Run

```bash
npm start
```

Open

```
http://localhost:3000
```
